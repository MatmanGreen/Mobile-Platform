# Software

## Overview

This directory contains the software components of the autonomous mobile robot.

The software stack will be based on ROS2 and will be responsible for:

- Motor control
- Sensor integration
- Robot localization
- SLAM
- Path planning

---

## Planned Technologies

- ROS2 Jazzy
- C++
- Gazebo
- OpenCV

---
