# Autonomous Mobile Robot

## Projektziel

Entwicklung eines autonomen mobilen Roboters mit:
- ROS2
- Computer Vision
- NEMA17 + TMC2209
- Gazebo Simulation
- SLAM

Ziel ist autonome Navigation in einfachen Innenräumen.
