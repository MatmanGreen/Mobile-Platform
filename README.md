# Autonomous Mobile Robot

## Project Goal

Development of an autonomous mobile robot using:
- ROS2
- Computer Vision
- NEMA17 + TMC2209
- Gazebo Simulation
- SLAM

The goal is autonomous navigation in simple indoor environments.
